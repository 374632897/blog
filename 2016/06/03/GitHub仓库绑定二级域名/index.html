GitHub仓库绑定二级域名

  • GitHub Pages

  1. 在仓库创建gh-pages分支

  2. 在该分支根目录下添加CNAME文件，文件内容是你希望指向的域名，需要注意的是不需要加协议，如期望网址是http://algorithms.noteawesome.com的话， 就直接写algorithms.noteawesome.com.


    在完成创建之后可以前往仓库设置查看是否设置成功

  3. 前往域名控制中心添加域名解析

  4. 慢慢等， 也可以用ping 'your site'来查看是否成功设置
